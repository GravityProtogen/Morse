Não deixe que os caracteres especiais ou pontos e traços te confundam.

Código Morse é um sistema para codificar letras e números em pontos e traços.
A duração de um traço é três vezes a duração de um ponto.
Cada ponto ou traço é seguido por um período de ausência de sinal, chamado de espaço, de mesma duração que um ponto.

O código Morse foi usado pela primeira vez para a comunicação via telégrafo.
O código Morse mais reconhecido é o SOS, que é usado para sinalizar uma emergência.
Além de digitar pontos e traços, você também pode sinalizar código Morse acendendo e apagando uma luz, soando uma buzina, batendo, entre outros.
Qualquer meio de transmissão de mensagens em Morse requer que o destinatário entenda esse código.
Se esse não for o caso, nossa ferramenta decifradora poderá ser usada para descobrir a senha, número secreto ou qualquer que seja a informação transmitida.
Como as pessoas raramente se comunicam via telégrafo, ou aprendem no exército, pouquíssimas memorizam códigos Morse.
Em vez disso, tornou-se uma forma de codificar coisas para fins especiais.
É uma linguagem útil de se saber, mesmo que você só queira aprender criptografia ou resolver criptogramas para se divertir.
